认证

所有 API 请求需在 Header 中携带 API 密钥：

Authorization: Bearer YOUR_API_KEY

API 密钥可在账户设置中生成。Pro 用户可创建多个密钥，Enterprise 用户可为团队成员分配独立密钥。

Base URL

https://api.sonicforge.ai/v1
POST /music/generate

创建新的音乐生成任务。任务为异步执行，返回任务 ID 后可通过状态接口查询进度。

请求参数

genre* string 音乐风格，如 "electronic", "jazz", "classical"
bpm integer 速度（60-200），默认 120
intensity integer 情感强度（0-100），默认 60
complexity integer 复杂度（0-100），默认 50
duration integer 时长（秒，10-120），默认 30
prompt string 自然语言描述，如 "夜晚城市漫游的电子音乐"
format string 输出格式："mp3", "wav", "flac"，默认 "mp3"

请求示例

{ "genre": "electronic", "bpm": 128, "intensity": 75, "complexity": 60, "duration": 45, "prompt": "深夜俱乐部的深邃浩室音乐，带有迷幻的合成器音色", "format": "wav" }

响应示例

200 OK
{ "task_id": "task_abc123xyz", "status": "processing", "estimated_time": 8, "created_at": "2026-06-21T10:30:00Z" }
GET /music/status/{task_id}

查询生成任务的状态。

响应示例（处理中）

200 OK
{ "task_id": "task_abc123xyz", "status": "processing", "progress": 65 }

响应示例（完成）

200 OK
{ "task_id": "task_abc123xyz", "status": "completed", "download_url": "https://api.sonicforge.ai/v1/music/download/task_abc123xyz", "duration": 45, "format": "wav", "copyright_id": "cert_xyz789" }
GET /music/download/{task_id}

下载已完成的音频文件。返回二进制音频数据。

Headers: Content-Type: audio/wav (或 audio/mpeg, audio/flac)

下载链接有效期为 24 小时，过期后需重新获取。

GET /genres

获取所有可用的音乐风格列表。

响应示例

200 OK
{ "genres": [ { "id": "electronic", "name": "Electronic", "sub_genres": ["Deep House", "Techno", ...], "bpm_range": [60, 180] }, ... ] }
GET /genres/{genre_id}

获取指定风格的详细信息，包括子风格、推荐参数和示例。

GET /history?page=1&limit=20

获取生成历史记录，支持分页。

查询参数

page integer 页码，默认 1
limit integer 每页数量（1-100），默认 20
genre string 按风格筛选
DELETE /history/{task_id}

删除指定的生成记录。删除后无法恢复，下载链接将失效。

响应

200 OK
{ "success": true }

速率限制

API 调用受速率限制保护：

  • Starter：10 次/分钟，30 次/小时
  • Pro：60 次/分钟，1000 次/小时
  • Enterprise：自定义限制

超出限制时返回 429 Too Many Requests，Header 中包含 Retry-After 字段。
